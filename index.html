Navigate to Downloads Folder

In PowerShell, navigate to your Downloads folder:

üìÅ Your project folder: C:\Users\%USERNAME%\Downloads
(This is where your migration files will be created)

Get Your Project Reference ID

Follow these steps to find your project ID:

  • Go to Supabase Dashboard
  • Select your project from the list
  • Click Settings (gear icon)
  • Click General in the left menu
  • Look for Project Reference (looks like: abcdefghijklmno)
  • Copy the entire reference ID

Run the Link Command

Make sure you're in your project folder, then run this command (replace with your actual project reference):

supabase link --project-ref YOUR_PROJECT_REFERENCE_ID

For example: supabase link --project-ref abcdefghijklmno

What Happens Next

  • CLI opens your browser for you to log in to Supabase
  • Confirm the project in the browser
  • Returns to PowerShell with a success message
